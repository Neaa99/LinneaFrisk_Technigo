What is HTTP?

HTTP stands for HyperText Transfer Protocol.

HyperText is a text that has hyperlinks that connect links with other links. The most common way to use HTTP is on the World Wide Web (WWW).

Transfer means that the information is sent and received.

For HTTP to work, rules are required, thus protocol, which refers to the agreement between two or more units on how the communication will take place.

HTTP is used when a web browser is retrieving information from a web server. These are usually HTML files or images, but can also be other types of files, such as video or zip files.
