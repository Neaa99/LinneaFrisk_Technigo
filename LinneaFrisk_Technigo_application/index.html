What is HTTP?

HTTP står för HyperText Transfer Protocol.

HyperText är en text som har hyperlänkar som kopplar ihop länkar med andra länkar. Den vanligaste sättet att använda HTTP på är på World Wide Web (WWW).

Transfer medför att informationen skickas och tas emot.

För att HTTP ska fungera krävs det att regler, alltså ett protokoll, som syftar på överrelskommelsen mellan två eller flera enheter om hur kommunikationen ska gå till. HTTP används när en webbläsare ska hämta information från en webbserver.

Vanligtvis handlar det om HTML-filer och bilder, men det kan även handla om andra typer av filer, som exempelvis video eller zip-filer.
